  • eScan Antivirus supply chain attack: Cybersecurity firm Morphisec reports that eScan users suffered a supply chain attack after hackers compromised a regional update server. Malicious updates distributed "Reload.exe," which disabled automatic updates via HOSTS file modification and established persistence.   • Instagram private profile data leak: Researcher Jatin Banga exposed a server-side authorization vulnerability in Instagram that leaked private account content to unauthenticated visitors. By analyzing HTML on mobile user-agents, the "polaris_timeline_connection" object was found to contain encoded CDN links to private photos and captions. Meta quietly patched the flaw shortly after disclosure. More info
  • Ivanti EPMM critical zero-days (CVE-2026-1281/1340): Ivanti released emergency updates for two critical flaws (CVSS 9.8) in Endpoint Manager Mobile allowing unauthenticated RCE via specialized HTTP GET requests.
