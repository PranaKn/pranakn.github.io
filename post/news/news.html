  • Microsoft Office Zero-Day (CVE-2026-21509): Microsoft issued emergency updates for a high-severity security feature bypass. The flaw involves improper handling of untrusted inputs, allowing attackers to bypass OLE mitigations via malicious files. Exploitation has been detected in the wild. More info
  • CISA Adds Three Exploited Flaws to KEV: CISA expanded its catalog to include CVE-2018-14634 (Linux kernel privilege escalation), CVE-2025-52691 (SmarterMail arbitrary file upload), and CVE-2026-23760 (SmarterMail admin password reset bypass). The SmarterMail flaws allow unauthenticated RCE or full account takeover. Users are urged to patch Linux kernels and upgrade SmarterMail to Build 9511 or later. More info
  • 800,000 Telnet Servers Exposed (CVE-2026-24061): A critical 11-year-old authentication bypass in the GNU InetUtils telnetd server allows remote root access. By passing a crafted "-f root" value during negotiation, attackers skip password verification. GreyNoise confirmed active exploitation immediately following the patch. Admins should upgrade to version 2.8 or disable Telnet. More info
  • "PackageGate" Bypasses JS Security: Researchers identified flaws in npm, pnpm, Bun, and vlt that bypass --ignore-scripts protections. By using Git-based dependencies and malicious .npmrc files, attackers can execute arbitrary code during installation. While most managers issued patches, npm maintains the behavior is intentional, leaving developers at risk of supply-chain RCE. More info
