🚨 Vulnerabilities

🔥 Ivanti Patches Critical RCE Vulnerability (CVE-2025-22457)
Ivanti has released patches for a critical vulnerability in Connect Secure, Policy Secure, and ZTA Gateways that allows unauthenticated remote code execution. Active exploitation by a China-nexus threat actor has been confirmed. Users are urged to update to version 22.7R2.6 or apply upcoming patches. CISA recommends threat hunting, factory resets, revoking credentials, and isolating compromised devices. Affected systems not updated by February 28 may already be compromised.
🔗 CISA Alert
🐍 Malicious PyPI Package 'disgrasya' Targets WooCommerce
A malicious PyPI package named 'disgrasya' was downloaded over 34,000 times before removal. It was used for credit card validation via WooCommerce stores using the CyberSource gateway. The script emulates shoppers, stealing checkout tokens and routing stolen card data to a fake CyberSource server controlled by the attacker. Mitigation strategies include blocking low-value orders, monitoring anomalies, applying CAPTCHAs, and rate-limiting sensitive endpoints.

🧩 WinRAR Flaw Bypasses Windows Mark of the Web (CVE-2025-31334)
CVE-2025-31334 is a medium-severity vulnerability in WinRAR (prior to version 7.11) that allows attackers to bypass Windows’ Mark of the Web (MotW) security warnings using specially crafted symlinks pointing to executable files. If exploited, it enables arbitrary code execution without triggering MotW alerts. The flaw has been fixed in version 7.11.

🔓 Apache Parquet Critical RCE Vulnerability (CVE-2025-30065)
A critical vulnerability (CVE-2025-30065, CVSS 10.0) has been disclosed in Apache Parquet’s Java library, allowing remote attackers to execute arbitrary code by tricking systems into loading crafted Parquet files. The flaw affects versions up to 1.15.0 and has been fixed in 1.15.1. It poses serious risks to data pipelines and analytics platforms handling untrusted files.

🔑 OIDC Misconfigurations in CI/CD Environments
Unit 42 researchers have uncovered critical security risks in how OpenID Connect (OIDC) is used within CI/CD environments, with specific issues found in CircleCI’s OIDC implementation. Misconfigurations can allow attackers to gain unauthorized access to protected resources. Organizations are urged to implement strict claim validation, avoid broad access patterns, and regularly audit their OIDC configurations.

🕵️ Threat Actors & Campaigns

📱 E-ZPass Phishing Campaign Surges
A phishing campaign targeting E-ZPass and other toll agencies is surging, using iMessage and SMS texts to trick victims into visiting fake websites and stealing personal and credit card details. These messages often use urgency to prompt action and bypass anti-spam filters by originating from random email addresses. Victims are urged not to reply and to verify toll balances directly with official websites.

🛡️ WordPress 'mu-plugins' Directory Abused for Malware
Threat actors are increasingly abusing WordPress’s 'mu-plugins' directory to execute persistent malicious code that evades detection. Sucuri researchers identified common payloads including redirectors to malicious sites, webshells for remote control, and spam injectors for SEO scams or fraud. These stealthy implants can persist unnoticed, making them a favored tactic for long-term WordPress site compromise.

🧑‍💻 North Korean Hackers Deploy BeaverTail Malware via npm Supply Chain
North Korean actors behind the “Contagious Interview” campaign have escalated npm-based attacks, distributing malicious packages like BeaverTail and others. These use obfuscation and C2 links to Lazarus infrastructure to infect developer systems under the guise of job interviews.
