  • Critical Flaws in Fluent Bit Expose Cloud and Kubernetes: Researchers uncovered five chainable vulnerabilities, including a path traversal **RCE** (CVE-2025-12972) and a stack buffer overflow (CVE-2025-12970), in the widely used **Fluent Bit** telemetry agent. These flaws allow attackers to bypass authentication, achieve **Remote Code Execution**, and manipulate logs, posing a severe supply chain risk to cloud and Kubernetes environments, including those on AWS. Users are strongly urged to update to Fluent Bit versions **4.1.1 or 4.0.12** immediately. More info

  • StealC Infostealer Delivered via Malicious Blender 3D Files: A Russian-linked campaign is covertly distributing the StealC V2 infostealer by embedding malicious Python scripts within Blender (.blend) 3D model files uploaded to marketplaces like CGTrader. If Blender's 'Auto Run Python Scripts' is enabled, the script executes, launching a multi-stage process that ultimately drops StealC V2, a formidable commodity stealer targeting data from **23+ browsers, 100+ crypto wallet extensions**, and communication clients. Users must disable the 'Auto Run' feature and treat 3D assets from untrusted sources as executable files. More info
  • Malicious VSCode Extension Drops Anivia Stealer: Cybersecurity firms swiftly removed a malicious, brandjacked **VSCode extension** named 'prettier-vscode-plus' that was designed to deploy the **Anivia Stealer**. The extension used a multi-stage, evasive attack by executing the stealer directly from memory to evade detection. Anivia Stealer, believed to be a rebranded ZeroTrace, targets and exfiltrates credentials, system metadata, and private data, including **WhatsApp chats**, highlighting the growing supply chain threat targeting developers. More info
  • RadzaRat Android RAT Evades All Major Anti-Virus Vendors: Researchers discovered **RadzaRat**, a new, highly effective **Android Remote Access Trojan (RAT)** disguised as a functional file manager. Actively sold on underground forums, the malware currently maintains **zero-detection**. Once installed, RadzaRat abuses Android's Accessibility Services to enable **full device control**, persistent keylogging, and data exfiltration, posing a significant, invisible threat due to its public availability and complete evasion capability. More info

Threat actors use zero-click exploits, social engineering, and malicious QR codes. A related zero-click exploit, **CVE-2025-21042 in Samsung devices** used to deliver LANDFALL spyware, has been added to the CISA KEV Catalog, mandating immediate remediation by federal agencies.   • ClickFix Attacks Distribute Rhadamanthys Infostealer via Fake Windows Updates: A new campaign of **ClickFix social engineering attacks** is distributing the **Rhadamanthys infostealer** using highly deceptive fake Windows update screens. The attack employs a sophisticated **steganographic loader**, which extracts shellcode hidden within the pixel data of PNG images to evade detection. Despite recent law enforcement action against Rhadamanthys, the lure pages remain active, demanding aggressive network monitoring and defense. More info
  • Black Friday Malvertising Uses Survey Reward Scams for Data Theft: An organized malvertising operation is deploying a large-scale **Survey Reward scam** during the Black Friday season, using over 100 unique domains to impersonate high-trust brands like **Walmart and Starlink**. The campaign uses invisible redirect chains and urgency to funnel victims into fake surveys that culminate in a checkout process, stealing full **credit card details and personal information** for a small "shipping fee." More info

Scams and phishing also increased significantly. Attackers are exploiting high-growth malicious carriers like TXT files (+180%) and DOC files (+118%). **Ransomware has returned at scale**, with 24% of organizations reporting an incident, driving over two-thirds of organizations to invest in AI-driven detection.
