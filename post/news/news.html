  • Critical Flaws in Versa, Zimbra, Vite, and ESLint: Several critical vulnerabilities have been disclosed across diverse enterprise and development platforms. Versa Concerto (CVE-2025-34026) suffers from improper authentication in its Traefik reverse proxy, exposing administrative endpoints. Synacor Zimbra (CVE-2025-68645) is vulnerable to PHP remote file inclusion. Vite (CVE-2025-31125) allows unauthorized file access when the dev server is exposed. Furthermore, the eslint-config-prettier package (CVE-2025-54313) was found to contain malicious code executing the node-gyp.dll malware on Windows during installation. More info
  • SmarterMail Auth Bypass (CVE-2026-23760): Threat actors are actively exploiting a critical authentication bypass in SmarterTools SmarterMail to hijack administrator accounts. The flaw resides in the force-reset-password API endpoint, allowing unauthenticated users to reset the system admin password via a crafted JSON request. Attackers have gained SYSTEM-level access and executed remote commands. Organizations must upgrade to Build 9511 immediately. More info
  • GNU InetUtils Telnetd Root Access (CVE-2026-24061): A CVSS 9.8 vulnerability has been identified in the telnet daemon (telnetd), existing since 2015. Attackers can gain root access by supplying a crafted USER environment variable (-f root), which skips authentication. GreyNoise reports active global exploitation. Administrators should disable telnetd or restrict network access. More info
  • Appsmith Account Takeover (CVE-2026-22794): Researchers identified a "password reset poisoning" flaw in the Appsmith low-code platform. The app trusts the client-provided Origin HTTP header to build reset URLs, allowing attackers to leak unique reset tokens to their own servers. This enables full account compromise and exposure of internal databases. Users should upgrade to version 1.93. More info
  • Pwn2Own Automotive 2026 Zero-Days: On the second day of the competition in Tokyo, researchers earned $439,250 by exploiting 29 unique zero-day vulnerabilities in EV chargers, infotainment systems, and Tesla systems. The total prize pool has reached nearly $1M. Vendors have a 90-day window to patch before public disclosure. More info

  • FortiGate SSO Exploitation: Arctic Wolf identified an automated campaign targeting Fortinet FortiGate devices via a suspected patch bypass (CVE-2025-59718). Attackers used the FortiCloud SSO feature to create rogue VPN accounts and steal firewall configs. Administrators are urged to disable FortiCloud SSO logins while awaiting FortiOS 7.4.11. More info
  • Osiris Ransomware Targets SE Asia: A novel strain called Osiris, distinct from the old Locky variant, targeted a SE Asian food operator using a hybrid ECC/AES encryption. The attack used a BYOVD maneuver with the POORTRY driver to disable security tools. Links to the INC ransomware group have been suggested based on Mimikatz builds and Rclone usage. More info
  • LLM-Augmented Phishing: Unit 42 detailed "LLM-Augmented Runtime Assembly" attacks that use generative AI to dynamically build phishing pages. By calling APIs like Gemini or DeepSeek at runtime, attackers generate unique, polymorphic malicious JavaScript that evades traditional network security analysis because the code is created in real-time. More info
  • Malicious PyPI Package 'sympy-dev': A fake package impersonating the SymPy library was found on PyPI. It acts as a downloader for an XMRig cryptocurrency miner, executing in memory using memfd_create to avoid disk footprints. While currently used for mining, the implant is a general-purpose loader. More info
