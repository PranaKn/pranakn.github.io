🛡️ Vulnerabilities

  • CISA adds SolarWinds, Ivanti, and Omnissa flaws to KEV: CISA has added three high-impact vulnerabilities to its Known Exploited Vulnerabilities catalog. The most critical, CVE-2025-26399 in SolarWinds Web Help Desk, is a deserialization flaw allowing unauthenticated RCE, currently used by ransomware groups. Also added were CVE-2026-1603 (Ivanti Endpoint Manager auth bypass) and CVE-2021-22054 (Omnissa Workspace ONE UEM SSRF). Federal agencies must patch these by late March 2026. More info
  • Overly permissive Salesforce Cloud configs expose data: Research highlights a growing risk where Salesforce Community and Experience Cloud misconfigurations grant "Guest" users excessive access to internal objects. Attackers are actively scanning for these errors to exfiltrate customer lists and internal communications. Organizations are urged to audit sharing rules as part of the shared responsibility model. More info
  • Zero-click Windows info disclosure bug: A critical vulnerability in Windows allows remote attackers to leak sensitive memory contents without user interaction. The flaw resides in network packet processing and can be triggered by sending a crafted request over the local network or internet. Microsoft has issued patches and stresses immediate updates. More info
  • March 2026 Patch Tuesday fixes 83 flaws: Microsoft addressed 83 vulnerabilities, including two zero-days exploited in the wild. Fixes include critical RCE bugs in Office and Exchange Server, and an elevation of privilege flaw allowing SYSTEM-level access. Administrators should prioritize these patches to prevent lateral movement. More info
  • Adobe releases massive update for 80 vulnerabilities: Adobe patched 80 flaws across Acrobat, Reader, and Photoshop. Many are critical memory corruption or out-of-bounds issues that could lead to RCE. While no active exploits are reported, the volume of patches necessitates rapid deployment. More info
  • Max-severity RCE in PAC4J security library: A critical flaw in the open-source PAC4J Java library allows remote code execution in specific configurations. Given its integration into numerous enterprise platforms, developers are urged to update immediately to secure the software supply chain. More info
  • SAP patches critical FS-QUO and NetWeaver flaws: Monthly security notes address high-impact vulnerabilities that could allow administrative access or arbitrary code execution. These updates are vital for protecting core business operations and financial processing environments. More info
  • "Fuzzing" AI judges bypasses LLM filters: Unit 42 research shows that systematically testing input variations can trick AI content moderators into approving malicious prompts. This reveals a weakness in relying solely on AI for security oversight, necessitating multi-layered defenses. More info
