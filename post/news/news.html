  • Gogs Self-Hosted Git Zero-Day RCE Exploited: A zero-day, **CVE-2025-8110**, is under active exploitation in Gogs, a self-hosted Git service, compromising over 700 internet-exposed instances. The flaw is an authentication bypass for a previous RCE bug, leveraging the `PutContents` API and symbolic links to overwrite the `.git/config` file's `sshCommand`, allowing remote arbitrary code execution for authenticated users with repository creation permissions. Admins must disable open-registration and place Gogs behind a VPN as no patch is immediately available. More info
  • .NET "SOAPwn" RCE Flaw Patched in Enterprise Apps: WatchTowr Labs revealed "SOAPwn," an "invalid cast vulnerability" in the .NET Framework that enables Remote Code Execution (RCE) by abusing Web Services Description Language (WSDL) imports and SOAP messages. The flaw can be exploited to achieve arbitrary file write via a `file://` path, leading to webshell uploads. Patches were released for Barracuda Service Center RMM (**CVE-2025-34392**, CVSS 9.8) and Ivanti Endpoint Manager (**CVE-2025-13659**, CVSS 8.8), but Microsoft chose not to fix the underlying framework issue. More info
  • Three Flaws in PCIe IDE Protocol Affect Intel/AMD CPUs: Three specification-level security flaws (**CVE-2025-9612, -9613, -9614**) were disclosed in the PCI Express (PCIe) Integrity and Data Encryption (IDE) protocol, affecting Base Specification Revision 5.0+. The flaws, which require physical or low-level access, undermine data protection, potentially causing information disclosure or privilege escalation by processing stale or incorrect data.   • Fortinet Patches Two Critical FortiCloud SSO Bypass Flaws: Fortinet released patches for 18 vulnerabilities, including two critical SAML authentication-bypass flaws, **CVE-2025-59718** and **CVE-2025-59719** (CVSS 9.1). These flaws, caused by improper cryptographic signature verification, allow an unauthenticated attacker to bypass FortiCloud Single Sign-On (SSO) login when the feature is enabled, granting full administrative control. Affected products include FortiOS, FortiWeb, FortiProxy, and FortiSwitchManager. More info
    • Google Fixes Gemini Enterprise Prompt Injection Flaw ("GeminiJack"): Google patched a zero-click, corporate data exfiltration vulnerability, dubbed **GeminiJack**, found in Gemini Enterprise and Vertex AI Search. The flaw was an architectural weakness allowing a type of **indirect prompt injection** exploiting the **Retrieval-Augmented Generation (RAG)** architecture. Attackers embedded hidden malicious instructions in shared Google Workspace content. When an employee searched, the RAG system retrieved the poisoned content, and Gemini executed a hidden instruction to exfiltrate data via a malicious external image tag in the response. More info

    • DroidLock Android Malware for Full Device Takeover: DroidLock is an evolving Android banking malware targeting Spanish-speaking users, distributed outside Google Play. It uses Device Admin and Accessibility Services to perform fraudulent activities, supporting 15 remote commands including device wiping and changing the lock PIN/password. Crucially, it achieves complete device takeover and credential theft via the **VNC sharing system**, stealing the device lock pattern via a screen overlay. More info
    • Agent Tesla RAT Spread via "One Battle After Another" Torrent: A multi-stage, fileless Living Off the Land (LOTL) attack is distributing the Agent Tesla Remote Access Trojan (RAT) via a malicious torrent for the film *One Battle After Another*. The attack uses a hidden command chain in a shortcut file to execute malicious **PowerShell scripts** embedded within a seemingly benign subtitle file, which runs the final payload entirely in memory to evade detection.     • ClickFix Deploys CastleLoader via Python Chain: A new campaign uses the ClickFix social engineering technique to deploy the CastleLoader malware via a Python-based delivery chain. The attack executes a command that launches a hidden `conhost.exe` process to download and execute a tar archive. A bundled Python interpreter reconstructs and decrypts CastleLoader shellcode in memory, which uses PEB Walking for stealthy API resolution before retrieving the final payload, commonly including infostealers (StealC, RedLine) and CastleRAT. More info
    • GhostFrame Phishing Kit Uses Hidden iFrames and BLOBs: GhostFrame is a new, highly evasive Phishing-as-a-Service (PhaaS) kit linked to over one million attacks. It utilizes a unique stealth architecture where the malicious activity is hidden inside an **iframe** loaded from constantly rotating subdomains. Credential forms within the iframe are hidden inside features designed for **image streaming (BLOBs)**, bypassing static content scanners. The kit also aggressively disables analysis tools like right-click and F12. More info
    • Rust-Based 01flip Ransomware Targets APAC Critical Infra: A new, financially motivated threat cluster (CL-CRI-1036) is deploying the multi-platform **01flip ransomware**, fully written in the Rust programming language. The attack chain targets critical infrastructure in the Asia-Pacific (APAC) region, starting with exploitation of older flaws like **CVE-2019-11580** (Atlassian Crowd RCE) and deploying the cross-platform adversary emulation tool **Sliver** for lateral movement before encrypting Windows and Linux machines.     • ClickFix Attack Uses ChatGPT/Grok to Deploy AMOS on macOS: A new, sophisticated ClickFix campaign utilizes **SEO poisoning** to promote shared conversations on legitimate LLM platforms (like ChatGPT and Grok) as top search results. These malicious conversations trick macOS users into copying and executing seemingly benign system troubleshooting commands in their Terminal, which actually download and install the **Atomic macOS Stealer (AMOS)** to harvest sensitive data like credentials and crypto wallets.     • StealC V2 Spread via Malicious Blender 3D Model Files: The **StealC V2 infostealer** is being distributed through free, malicious **.blend** 3D model files hosted on legitimate marketplaces. The attack leverages Blender's standard, but dangerous, **Auto Run Python Scripts** feature to automatically execute an embedded Python script upon file opening. This script initiates a multi-stage, fileless attack that installs the powerful infostealer, targeting over 23 browsers and 15 crypto wallets. More info
    • New SVG-Based Clickjacking Bypass: A security researcher unveiled a new clickjacking technique that exploits a loophole in Scalable Vector Graphics (SVG) filters and Cascading Style Sheets (CSS) to bypass **cross-origin isolation**. The attack leverages SVG filters (specifically `feBlend` and `feComposite`) to construct **logic gates** capable of processing pixel data from a cross-origin framed document, effectively allowing arbitrary computation on data protected by the Same-Origin Policy. The flaw affects Chrome and Firefox. More info
