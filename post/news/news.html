  • Exploited "MongoBleed" (CVE-2025-14847) leaks secrets: A critical information disclosure vulnerability affecting over 80,000 MongoDB instances is under active exploitation. The flaw (CVSS 8.7) stems from improper length handling in zlib-compressed headers, allowing unauthenticated attackers to exfiltrate database passwords, AWS keys, and session tokens via uninitialized heap memory leaks. MongoDB urges an immediate upgrade or disabling zlib compression. More info
  • "LangGrinch" (CVE-2025-68664) hits LangChain Core: This critical serialization injection flaw (CVSS 9.3) allows attackers to use prompt injection to instantiate unsafe objects. This can lead to remote code execution or the theft of sensitive secrets like OpenAI API keys. Users are advised to upgrade to versions 1.2.5 or 0.3.81, which disable automatic secret loading. The malware stole contact lists and session tokens from over 56,000 users and hijacked the pairing process for persistent account access. Victims must manually unlink unrecognized devices in WhatsApp settings to secure their accounts. More info
