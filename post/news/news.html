🚨 Vulnerabilities

🔥 Apache Parquet RCE Vulnerability (CVE-2025-30065)
A critical remote code execution vulnerability with a CVSS score of 10.0 has been discovered in Apache Parquet versions up to 1.15.0. Attackers can exploit deserialization in the parquet-avro module by tricking systems into importing specially crafted Parquet files. Administrators are urged to upgrade to version 1.15.1 immediately and avoid untrusted Parquet files in the meantime.

🔓 Ivanti Connect Secure RCE Vulnerability (CVE-2025-22457)
Ivanti has patched a critical remote code execution vulnerability in Connect Secure, exploited by the China-linked espionage group UNC5221 since at least mid-March 2025. Administrators should upgrade immediately, monitor with Ivanti's ICT, and audit for compromise, especially among unpatched appliances.

🧪 PyTorch Lightning Deserialization Vulnerabilities
PyTorch Lightning versions 2.4.0 and earlier contain multiple high-risk deserialization vulnerabilities due to unsafe use of torch.load() and Python’s pickle, potentially allowing arbitrary code execution when loading model files. Users are urged to only load models from trusted sources, enforce safe loading practices, and monitor for updates from Lightning AI.

📱 Google Quick Share Vulnerabilities Bypassed
SafeBreach researchers revealed they had bypassed patches for two critical Google Quick Share vulnerabilities (CVE-2024-38272, CVE-2024-38271) that were part of an earlier disclosed 10-bug exploit chain dubbed “QuickShell.” The findings highlight how combining minor vulnerabilities can create critical threats.

🖥️ Cisco Meraki and ECE DoS Vulnerabilities
Cisco has patched two high-severity denial-of-service vulnerabilities: CVE-2025-20212, affecting Meraki MX/Z devices via a flaw in the AnyConnect VPN server, and CVE-2025-20139, impacting Cisco’s Enterprise Chat and Email (ECE) through unauthenticated malicious chat requests. Patching is strongly advised.

📄 GitHub Supply Chain Attack Traced to Leaked Token
A sophisticated supply chain attack on GitHub that ultimately targeted Coinbase has been traced back to a single Personal Access Token (PAT) leaked in a SpotBugs workflow in November 2024. Security teams are advised to rotate secrets and use commit hashes over tags for workflows.

🐱 HellCat Ransomware Group Emerges
HellCat is a relatively new ransomware-as-a-service (RaaS) group that emerged in mid-to-late 2024, known for double extortion attacks—encrypting systems and stealing data to demand ransom. There's currently no public decryptor, and security experts recommend standard ransomware defenses.
