  • Urban VPN Proxy Steals AI Chatbot Data: A widely used Google Chrome and Edge extension, **Urban VPN Proxy**, with over eight million installs, was caught harvesting all user prompts and responses from major AI chatbots (e.g., ChatGPT, Gemini). Version 5.5.0 injected a JavaScript executor to override browser APIs, capturing and exfiltrating non-anonymized conversation data, including potentially sensitive details, to remote servers for monetization.   • Atlassian Patches Maximum-Severity XXE Flaw (CVE-2025-66516): Atlassian released a critical update addressing dozens of flaws, most notably a **CVSS 10.0** XML External Entity (XXE) injection vulnerability in the Apache Tika content analysis toolkit used by Jira, Confluence, and Bamboo. The flaw allows an unauthenticated remote attacker to embed a malicious XFA file in a PDF, leading to information disclosure, Server-Side Request Forgery (SSRF), or potentially Remote Code Execution (RCE) by accessing internal resources. The fix is in `tika-core` version 3.2.2 or later. More info

  • Operation MoneyMount-ISO Targets Russian Finance: A new phishing campaign, **Operation MoneyMount-ISO**, is actively targeting Russian-speaking finance and payroll sectors with the **Phantom information-stealing malware**. The attack bypasses email defenses by using a ZIP archive containing a malicious ISO file disguised as a "Confirmation of Bank Transfer." When opened, the ISO auto-mounts and executes a staged payload that injects Phantom Stealer into memory, exfiltrating a wide range of sensitive data via multiple channels, including Telegram bots and Discord webhooks.   • VolkLocker Ransomware Flaw Exposed: The pro-Russia hacktivist group **CyberVolk** relaunched its Ransomware-as-a-Service (RaaS) offering, **VolkLocker** (aka CyberVolk 2.x), supporting both Windows and Linux systems with sophisticated Telegram automation for C2. However, a critical implementation flaw was discovered: the **AES-256 master encryption key is written in plaintext** to a file in the victim's `%TEMP%` folder, allowing victims to self-decrypt their files without paying the ransom.   • SoundCloud Breach Confirmed by ShinyHunters: Audio streaming platform SoundCloud confirmed a recent security breach of an ancillary service dashboard, attributed to the **ShinyHunters extortion gang**, which disrupted VPN connectivity and caused temporary DoS attacks. The breach impacted about 20% of users (28 million accounts), exposing only email addresses and publicly visible profile information; no sensitive data like financial details or passwords were stolen. The company has blocked unauthorized access and implemented stronger security controls. More info

  • Google Discontinues Dark Web Report Feature: Google is shutting down its "dark web report" security tool, which notified users of compromised personal information found in illicit data dumps. The company cites user feedback that the reports did not provide "helpful next steps." Monitoring ceases on January 15, 2026, and the full report data will be deleted by February 16, 2026. Google encourages users to use integrated tools like Password Manager, Password Checkup, 2-Step Verification, and the "Results about you" feature. More info
  • Microsoft Updates Break Message Queuing (MSMQ): Microsoft confirmed that its December 2025 security updates (e.g., KB5071546) are causing widespread failures in the **Message Queuing (MSMQ) service** on Windows 10 and Server 2016/2019. The issue stems from security model changes that now require non-administrative MSMQ users to have write access to the typically restricted `C:\\Windows\\System32\\MSMQ\\storage` folder, leading to application errors. A fix timeline is unavailable, forcing administrators to consider a risky update rollback. More info
