🛡️ Vulnerabilities

  • Qualys - CrackArmor Flaws in AppArmor: Qualys researchers discovered "CrackArmor," critical vulnerabilities in the Linux AppArmor security module. These flaws allow local attackers to bypass security profiles and escalate privileges to root level due to improper system call handling. Given AppArmor's role in sandboxing for Ubuntu and Debian, this poses a major risk to cloud and container environments. More info
  • Veeam Critical RCE Flaws: Veeam issued an urgent warning regarding critical vulnerabilities in its backup servers allowing unauthenticated remote code execution (RCE). These flaws could lead to full control over backup infrastructure and data deletion. Administrators are urged to patch Backup & Replication instances immediately. More info
  • Ally WordPress Plugin Flaw: A critical unauthenticated privilege escalation vulnerability in the "Ally" WordPress plugin has exposed over 200,000 websites to potential takeover. Attackers can gain admin access via crafted requests to inject scripts or steal data. More info
  • Apple Patches Coruna Exploit Kit: Apple released emergency updates for older iOS versions to fix flaws exploited by the "Coruna" kit. This kit allowed zero-click remote code execution for targeted surveillance. The patch aims to protect users on older hardware who are targets of commercial spyware. More info
