  • Critical n8n RCE Flaws (CVE-2026-25049): Researchers identified flaws in the n8n workflow platform allowing authenticated users to bypass JavaScript sanitization and escape the sandbox. This follows the "Ni8mare" flaw (CVE-2026-21858) currently seeing massive scanning activity. Exploitation enables theft of API keys and database credentials. n8n released versions 2.5.2 and 1.123.17 to address the issues. More info
  • VMware ESXi Sandbox Escape (CVE-2025-22225): CISA confirmed ransomware gangs are actively exploiting this high-severity flaw. Originally used by Chinese-speaking actors as a zero-day in 2024, it allows arbitrary kernel writes to escape a VM to the host. CISA has added it to the KEV catalog, urging immediate patching of affected ESXi environments. More info
  • Orchid Security tackles "Identity Dark Matter": Orchid introduced an Identity Control Plane to find unmanaged authentication logic and embedded credentials within application code. Using LLM-driven observability rather than simple configuration audits, the tool extracts "identity as coded" to provide real-time governance of both managed and unmanaged environments. More info
  • Google Looker "LookOut" Vulnerabilities: Tenable disclosed critical bugs in self-hosted Looker deployments. The "LookOut" chain allows for RCE and full server control, while a second flaw enables extraction of internal management databases and secrets. Managed cloud services are secured, but self-hosted admins must update to 25.12.30+ or 24.18.209+ immediately. More info

üéØ Adversaries

  • LLMjacking in AWS Intrusion: Sysdig detailed a cloud breach where an attacker gained admin control in under 10 minutes. The actor used LLMs to automate reconnaissance and privilege escalation via Lambda code injection. Evidence of AI assistance included "LLMjacking" of Amazon Bedrock to generate scripts with Serbian-coded comments and hallucinations of non-existent repositories. More info
  • Global SystemBC Botnet Expansion: Silent Push uncovered a SystemBC operation with over 10,000 infected IPs, including government infra in Vietnam. A novel Perl-based Linux variant with zero AV detection was identified. The botnet serves as a precursor to ransomware, acting as a SOCKS5 proxy relay to mask malicious traffic. More info
  • Mustang Panda targets Diplomats: The China-linked group used fake U.S. diplomatic briefings to target government officials in Asia and Eastern Europe. Utilizing DLL search-order hijacking, they deployed DOPLUGS (a PlugX downloader) for quiet, persistent data collection during sensitive international coordination periods. More info
  • ShadowSyndicate Infrastructure Growth: Group-IB linked dozens of servers through reused SSH fingerprints, identifying a cluster supporting major ransomware groups like Cl0p and Black Basta. ShadowSyndicate appears to function as an Initial Access Broker or bulletproof host, transferring servers between internal clusters to hide ownership. More info

üìà Trends

  • Spear-Phishing via .scr Files: ReliaQuest identified a trend of using Windows screensaver files to bypass execution controls. These Portable Executables are being used to deploy JWrapper (an RMM tool) for persistent remote access, often disguised as business invoices or project summaries hosted on public clouds. More info
  • AI Chrome Extensions Privacy Risks: A study by Incogni found that 52% of AI-powered Chrome extensions collect user data, with 42% utilizing "scripting" permissions that allow them to read and modify browser content. Tools for programming and transcription were flagged as the most invasive, potentially exposing PII for millions of users. More info
  • Broken Nitrogen Ransomware Decryptor: Coveware identified a programming flaw in Nitrogen's VMware ESXi encryptor that makes data recovery impossible. The malware incorrectly loads a QWORD variable into memory, overwriting the first four bytes of the public key. Victims are warned that paying the ransom is entirely futile. More info
