  • Firefox Wasm Engine Flaw Exposed 180M Users (CVE-2025-13016): A high-severity stack buffer overflow in the **Firefox WebAssembly (Wasm) engine** was silently exposed for six months, affecting over 180 million users. The flaw, a mistake in pointer arithmetic within the Garbage Collection mechanism, could lead to **arbitrary code execution** by forcing memory corruption on a malicious webpage. Mozilla patched the issue on November 11, 2025, in versions Firefox 145 and Firefox ESR 140.5. More info
  • IoT Privacy Risk in Aircraft Cabins: A study reveals that the primary privacy risk in **aircraft cabin IoT devices** occurs after data is received by authorized devices. The publish-subscribe network architecture allows devices to view entire messages, often including raw sensor readings that disclose **private passenger body movements** or proprietary vendor data. Researchers advocate for on-device protection methods like differential privacy or secret sharing, as the network structure, not the privacy logic, causes the exposure. More info
  • "HashJack" Indirect Prompt Injection Targets AI Browser Assistants: The novel **HashJack attack** exploits the URL fragment (the section after a '#') to hide malicious instructions, targeting AI browser assistants like Copilot and Gemini. Since the fragment is never sent to the server, this technique **weaponizes legitimate websites** to trick the AI, bypassing traditional network and server-side defenses to command harmful actions like data exfiltration. Microsoft and Perplexity have applied fixes. More info

  • FlexibleFerret Operators Deploy Multi-Stage macOS Malware: Jamf Threat Labs attributed a sophisticated macOS campaign to **FlexibleFerret**, using a multi-stage infection chain for persistence and credential theft. The malware uses a script that adapts based on the chip (arm64/Intel), uses a LaunchAgent for persistence, and launches a **decoy application** that mimics legitimate Chrome prompts to steal credentials, exfiltrating them to Dropbox. The final stage deploys the **CDrivers Go-based backdoor** for system control and automated credential theft. More info
  • ToddyCat APT Evolves to Steal Corporate Email Data: The **ToddyCat APT** group is using new TTPs to steal corporate email data. They deploy a PowerShell variant of their **TomBerBil malware** on domain controllers to remotely steal browser data and credentials, which are then decrypted using captured DPAPI keys. A custom C++ tool, **TCSectorCopy**, bypasses restrictions to read Outlook OST files sector-by-sector, while other tools harvest Microsoft 365 OAuth 2.0 access tokens for persistent external mail access.   • JackFix Campaign Uses Fake Windows Update Lure: The **JackFix campaign** is an evolution of the ClickFix social engineering attack, using fake adult websites as a lure to hijack the victim's browser screen and display a convincing, full-screen **"critical" Windows security update prompt**. This tricks the user into manually executing malicious commands that launch an MSHTA payload, fetching an obfuscated PowerShell script. The script gains admin privileges, sets Windows Defender exclusions, and deploys up to eight different malware payloads, including infostealers like Rhadamanthys, Vidar, and RedLine. More info
  • "Dark LLMs" Lower Cybercrime Barrier: Specialized, malicious Large Language Models (LLMs) like the commercial **WormGPT 4** (available for $220) and the free, open-source **KawaiiGPT** are significantly lowering the technical barrier to entry for cybercrime. WormGPT 4 can generate fully functional **AES-256 ransomware PowerShell scripts**, while KawaiiGPT creates convincing spear-phishing emails and scripts for lateral movement, automating critical steps in the attack process.   • Tor Project Adopts CGO Encryption Algorithm: The Tor Project is replacing its outdated **tor1 relay encryption algorithm** with **Counter Galois Onion (CGO)** to bolster the network against modern traffic-interception and tagging attacks, enhancing user anonymity. CGO uses wide-block encryption, tag chaining to block tagging, and updates encryption keys after every cell for immediate forward secrecy, addressing key weaknesses in the old algorithm. More info
