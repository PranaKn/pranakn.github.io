# 18/11/2025 **VULNERABILITIES** Researchers uncovered that Cursor, a fast-growing AI-powered developer environment, is vulnerable to JavaScript injection attacks that let adversaries hijack its internal browser and steal credentials. Knostic found Cursor lacks integrity checks that tools like VS Code perform, making its runtime components easier to tamper with. By abusing a malicious MCP server—whose broad permissions let it modify components—attackers can overwrite login pages, inject arbitrary HTML/JS, and force every new browser tab to run attacker-controlled code. Cursor says this is inherent to how the tool works, not a fixable bug. The research highlights rising supply-chain risks in AI-assisted development and urges strict vetting of extensions and MCP servers. - https://www.darkreading.com/vulnerabilities-threats/cursor-issue-credential-stealing-attacks HiddenLayer researchers uncovered **EchoGram**, a major flaw in LLM guardrails affecting models like GPT-5.1, Claude, and Gemini. Guardrails typically block harmful prompts using either LLM-as-a-judge or classification filters, but EchoGram bypasses both by inserting “flip tokens”—nonsensical words or sequences missing from guardrail training data. These tokens trick the defence layer into reversing its decision, allowing malicious prompts through or falsely flagging safe queries. A single token such as “=coffee” was enough to bypass protections in tests. The technique can also generate alert fatigue by triggering false positives. HiddenLayer warns attackers could replicate the method within months, risking misuse in sensitive sectors like finance and healthcare. - https://hackread.com/echogram-flaw-bypass-guardrails-major-llms/ **ADVERSARIES** Cisco Talos reports that Kraken, a Russian-speaking ransomware group emerging from the former HelloKitty cartel, conducted major big-game hunting attacks in August 2025. The group exploits SMB flaws for initial access, uses Cloudflare for persistence, and leverages SSHFS to steal data before encrypting systems across Windows, Linux, and ESXi. Kraken’s new ransomware strain benchmarks each victim machine to choose the fastest, least-detectable encryption method. Its leak site promotes a new forum, “Last Haven,” backed by former HelloKitty operators. Attacks involve credential theft, RDP access, and multi-threaded encryptors, with ransoms around $1M. The issue reportedly remained unpatched for more than 15 months after a HackerOne report was marked “Informative.” DoorDash ultimately fixed the flaw but banned the researcher, accusing them of extortion after they demanded payment; the researcher says the company ignored the issue until pressured. Unlike earlier static scanners, mcp-scan can act as a **dynamic proxy**, sitting between AI agents and MCP servers to log, analyze, and block risky tool calls. It detects prompt injection, insecure configurations, over-privileged tools, and malicious flows. YAML guardrails allow custom blocking rules, while detailed audit logs support incident response and SIEM ingestion. Red teams can use it to reveal hidden capabilities and build attack chains; defenders can baseline agent behavior and enforce strict policies on high-risk tools. - https://github.com/invariantlabs-ai/mcp-scan
