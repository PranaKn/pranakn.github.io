🛡️ Vulnerabilities

  • Critical Nginx UI flaw (CVE-2026-27944) exposes server backups: A vulnerability in Nginx UI versions prior to 2.3.3 allows unauthenticated access to the /api/backup endpoint. Attackers can retrieve full system backups containing encryption keys, SSL private keys, and session tokens, leading to total server compromise. Administrators are urged to update to version 2.3.3 immediately. More info
  • Anthropic’s Claude Opus 4.6 identifies 22 Firefox bugs: In a collaboration with Mozilla, Anthropic used its AI model to find 22 vulnerabilities in two weeks. Fourteen were high-severity, including complex use-after-free and JIT miscompilation issues that traditional tools missed. While excellent at discovery, the AI struggled to create functional exploits, succeeding in only two cases. More info
  • CISA warns of Apple flaw (CVE-2026-20700) in spyware attacks: A memory corruption issue in Apple's Dynamic Link Editor (dyld) is being exploited in the wild for arbitrary code execution on iOS, macOS, and visionOS. The flaw is linked to spyware and crypto-theft campaigns. CISA has added it to the KEV catalog, urging immediate patching to protect mobile infrastructure. More info

🎯 Adversaries

  • Massive GitHub operation spreads BoryptGrab stealer: Over 100 malicious repositories are abusing GitHub’s infrastructure to distribute the BoryptGrab infostealer. Disguised as legitimate developer tools, the malware harvests browser data, crypto wallets, and system metadata. The campaign highlights the growing trend of poisoning the open-source ecosystem via automated scripts. More info | More info
  • Termite ransomware linked to ClickFix/CastleRat lures: The Termite group is using "ClickFix" tactics, injecting malicious JS into websites to show fake browser update alerts. These lures trick users into running PowerShell commands that bypass security features to deploy infostealers and eventually Termite ransomware. More info
  • RedAlert spyware targets Israeli citizens: A trojanized version of the official rocket warning app is being spread via SMS phishing. The fake app provides real-time alerts as a cover while stealing SMS, contacts, and GPS data in the background. It uses reflection and proxy hooks to evade Android security checks. More info
  • Fake Google Meet updates deliver infostealers: A new ClickFix campaign uses a "Fix It" button on fake Google Meet pages to copy malicious PowerShell commands to the user's clipboard. Manually executing these commands installs malware that provides full remote control over the victim's PC. More info
  • InstallFix targets developers with fake Claude Code guides: Attackers are cloning Anthropic’s Claude Code CLI installation pages via Google Ads. The fake sites provide modified terminal commands that download the Amatera infostealer. The malware communicates via legitimate CDN IPs to evade detection. More info
  • Microsoft warns of widespread ClickFix campaigns: Threat actors are compromising legitimate websites to display overlays claiming technical errors. Users are instructed to run "fix" scripts that install RATs. This social engineering tactic is specifically designed to bypass modern browser protections. More info
  • Iranian MuddyWater hits US bank and airport: The Seedworm group (MuddyWater) used a new backdoor called "Dindoor," leveraging the Deno JavaScript runtime. The campaign targeted a US bank, airport, and defense firm, likely as retaliation for regional military escalations. More info

📈 Trends

  • Hackers abuse .arpa DNS and IPv6 to evade defenses: Threat actors are hosting phishing pages on .arpa subdomains and using IPv6 tunnels to bypass traditional IP-based reputation filters. These techniques allow redirection chains to persist undetected by standard security monitoring. More info
  • Microsoft: AI integrated into every attack stage: A new report highlights how attackers use GenAI for reconnaissance, vulnerability discovery, and "vibe-hacking" to mimic authentic human behavior. This shift necessitates a move toward AI-driven defensive security operations. More info
  • Ransomware groups intensify targeting of education sector: School districts and universities remain top targets due to limited budgets and sensitive data. Double-extortion schemes involving student records are common, emphasizing the need for MFA and network segmentation. More info
  • Spyware disguised as emergency alert apps: Malicious payloads are increasingly hidden behind branding for official emergency alerts. These apps exploit wartime or disaster-related fears to trick users into granting high-level permissions for data exfiltration. More info
  • Fake CleanMyMac site distributes SHub Stealer: macOS users are being lured to a fraudulent site that installs SHub Stealer via terminal commands. The malware backdoors Electron-based crypto wallets like Exodus and Ledger Live to steal recovery phrases. More info
