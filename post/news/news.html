Cybersecurity Newsfeed

📅 23/02/26

🛡️ Vulnerabilities

  • Roundcube Webmail Critical RCE (CVE-2025-49113): Version 1.6.10 and earlier are susceptible to a critical RCE flaw stemming from insecure deserialization in the upload.php component. Authenticated attackers can exploit unvalidated URL parameters to execute arbitrary code. CISA added this to the KEV catalog, noting it was weaponized within 48 hours of disclosure. More info
  • Grandstream GXP1600 Series Root Access (CVE-2026-2329): A critical stack-based buffer overflow in the web API of these VoIP phones allows unauthenticated remote attackers to gain root privileges. This enables silent call interception and internal network pivoting. Users should update to firmware 1.0.7.81. More info
  • dYdX Crypto Exchange Supply Chain Attack: Attackers hijacked developer credentials to publish trojanized npm and PyPI packages. The malicious code exfiltrates wallet seed phrases and device fingerprints, while the PyPI variant deploys a RAT for persistent surveillance of developer environments. More info
  • Google Android 2025 Security Review: Google reported rejecting 1.75M apps and banning 80,000 malicious developer accounts in 2025. AI-driven defenses now scan 350 billion apps daily, identifying 27M malicious sideloaded applications and introducing native tapjacking protections in Android 16. More info

🎯 Adversaries

  • Arkanix Stealer: AI-Generated Malware-as-a-Service: Kaspersky identified a modular stealer likely developed using LLMs. It targets 22 browsers to harvest credentials and uses "Chrome Elevator" to bypass App-Bound Encryption. The project was abruptly terminated after two months, suggesting an AI experiment. More info
  • Pulsar RAT delivered via Steganography: A malicious NPM package ("buildrunner-dev") uses a 1,600-line obfuscated script and a PNG image to hide malicious code within RGB pixel values. It deploys the Pulsar RAT via process hollowing to gain full administrative control of Windows systems. More info
  • Intellexa’s Predator Spyware Bypasses iOS Privacy Icons: Predator now hooks iOS kernel-level processes to suppress the orange/green privacy indicators (mic/camera). This allows the spyware to stream audio and video secretly while making the device appear inactive to the user. More info
  • TrustConnect MaaS rebranding to DocConnect: Proofpoint uncovered a RAT operation using AI-generated content and EV certificates to pose as legitimate RMM software. Despite infrastructure takedowns, the operators quickly rebranded from TrustConnect to DocConnect to maintain resilience. More info
  • PromptSpy: First Android Malware Using Gemini AI: Distributed via fake banking sites, PromptSpy uses Google’s Gemini AI to analyze screen layouts in real-time. It generates device-specific instructions to maintain persistence and bypass UI limitations across different Android versions. More info

📈 Trends

  • Starkiller PaaS Bypasses MFA: This new Phishing-as-a-Service platform uses headless Chrome instances to act as a real-time man-in-the-middle. It relays live content from brands like Microsoft to capture session cookies and MFA tokens, neutralizing traditional account security. More info
  • ClickFix Campaigns Delivering MIMICRAT: Attackers are using compromised sites to show fake Cloudflare verification pages. Victims are tricked into running PowerShell commands that deploy MIMICRAT, a C++ RAT that supports token impersonation and SOCKS5 tunneling. More info
  • DDoS Surge: 168% Increase in 2025: Radware reports a massive spike in DDoS activity, with the tech sector bearing 45% of the load. Modern attacks are shorter and more intense, often coordinated by pro-Russian hacktivist groups via Telegram. More info
  • Fake Windows 11 "25H2" Update Malvertising: Facebook ads are leading users to clones of Microsoft's download page. The installer deploys "Lunar," an Electron-based infostealer that targets browser credentials and crypto wallets while using geofencing to avoid detection. More info
  • FBI Warning: ATM Jackpotting Surge: Over $20 million was stolen via 700 jackpotting incidents in 2025. Attackers use physical keys and Ploutus malware to hijack the ATM's cash dispenser, a technique frequently linked to organized crime syndicates. More info

💥 Breaches & Leaks

  • French Ministry of Economics / FICOBA Breach: Attackers used stolen credentials to access the national bank account registry, compromising 1.2 million records including IBANs, names, and tax IDs. More info
  • PayPal Data Leak in Loan App: A software coding error exposed sensitive PII (including SSNs and dates of birth) for customers of the PayPal Working Capital loan application over a six-month period. Affected users are being offered credit monitoring. More info
  • UMMC Ransomware Attack: The University of Mississippi Medical Center was forced to close 35 clinical locations after a ransomware attack disabled their Epic EMR system. Emergency services remain open under manual "pen-and-paper" protocols. More info
