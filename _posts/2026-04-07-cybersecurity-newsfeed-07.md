# Cybersecurity Newsfeed

## 📅 07/04/26

## 🛡️ Vulnerabilities

- **BlueHammer Windows Zero-Day Leak**: A functional zero-day exploit targeting the Windows kernel has been publicly leaked by a disgruntled researcher following a bug bounty dispute. The exploit allows for local privilege escalation (LPE). Microsoft has acknowledged the flaw but a formal patch is still pending. [More info](https://www.bleepingcomputer.com/news/security/disgruntled-researcher-leaks-bluehammer-windows-zero-day-exploit/)

- **CISA Expands KEV Catalog**: CISA has added a new vulnerability to its Known Exploited Vulnerabilities (KEV) catalog due to active exploitation in the wild. Federal agencies are mandated to remediate the flaw within a strict timeframe to prevent unauthorized access. [More info](https://www.cisa.gov/news-events/alerts/2026/04/06/cisa-adds-one-known-exploited-vulnerability-catalog)

## 🎯 Adversaries

- **DPRK Targets GitHub for C2 Infrastructure**: North Korean threat actors are utilizing malicious LNK files and GitHub to host command-and-control payloads. This shift toward using legitimate cloud platforms helps mask PowerShell-based data exfiltration and bypasses traditional security perimeters. [More info](https://securityaffairs.com/190413/uncategorized/phishing-lnk-files-and-github-c2-power-new-dprk-cyber-attacks.html)

- **Iranian Password-Spraying Campaign**: An Iranian-linked actor is conducting global password-spraying attacks against telecommunications, government, and defense sectors. The campaign exploits weak MFA and password policies to gain initial access for lateral movement. [More info](https://thehackernews.com/2026/04/iran-linked-password-spraying-campaign.html)

- **Medusa Ransomware Zero-Day Tactics**: Microsoft has linked a specific Medusa ransomware affiliate to the use of zero-day vulnerabilities for rapid corporate network entry. This indicates a high level of funding and technical sophistication in the RaaS ecosystem. [More info](https://www.bleepingcomputer.com/news/security/microsoft-links-medusa-ransomware-affiliate-to-zero-day-attacks/)

- **Storm-1175 High-Velocity Attacks**: The threat actor Storm-1175 is scanning for vulnerable web-facing assets to deploy Medusa ransomware. Their "high-tempo" operations complete full-scale encryption within hours of initial compromise. [More info](https://www.microsoft.com/en-us/security/blog/2026/04/06/storm-1175-focuses-gaze-on-vulnerable-web-facing-assets-in-high-tempo-medusa-ransomware-operations/)

- **Geopolitical Missile Alert Phishing**: A sophisticated campaign is targeting users in the US, Israel, and Iran with fake missile alert notifications. The goal is to harvest Microsoft credentials via perfectly mirrored fraudulent login pages. [More info](https://hackread.com/missile-alert-phishing-iran-us-israel-microsoft-logins/)

## 📈 Trends

- **AI-Powered Web Defense for CMS**: Cloudflare has launched "emdash," an AI tool designed to protect WordPress sites. It uses machine learning to block SQL injections and zero-day exploits by analyzing behavioral anomalies in real-time. [More info](https://hackread.com/cloudflare-wordpress-ai-powered-emdash-cms/)

- **Microsoft Deprecates Diagnostic Tools**: Microsoft has officially removed the Support and Recovery Assistant (SaRA) from Windows, moving toward integrated, web-based diagnostics within the Settings app. [More info](https://www.bleepingcomputer.com/news/microsoft/microsoft-removes-support-and-recovery-assistant-from-windows/)

## ⚖️ Law Enforcement

- **BKA Unmasks REvil Operators**: The German Federal Criminal Police Office (BKA) has successfully identified two key operators of the REvil ransomware gang responsible for over 130 attacks in Germany. [More info](https://securityaffairs.com/190401/cyber-crime/bka-unmasks-two-revil-ransomware-operators-behind-130-german-attacks.html)
