# Cybersecurity Newsfeed

## 📅 29/04/26

## 🛡️ Vulnerabilities

- **Critical GitHub RCE (CVE-2026-3854)**: A flaw in repository metadata processing allows remote code execution by bypassing security sandboxes. GitHub has released mitigations, but users are urged to update internal Enterprise instances. [More info](https://securityaffairs.com/191434/security-cve-2026-3854-github-flaw-enables-remote-code-execution.html)

- **LiteLLM Pre-Auth SQL Injection**: Attackers are actively exploiting an open-source proxy for LLM providers, allowing arbitrary SQL commands without authentication. [More info](https://www.bleepingcomputer.com/news/security/hackers-are-exploiting-a-critical-litellm-pre-auth-sqli-flaw/)

- **Microsoft Zero-Day Exploitation**: Microsoft confirmed APT groups are actively using a new zero-day for remote code execution across the ecosystem. Emergency mitigations and signatures have been released. [More info](https://thehackernews.com/2026/04/microsoft-confirms-active-exploitation.html)

- **CISA Adds Two Flaws to KEV**: Two new vulnerabilities affecting enterprise software have been added to the Known Exploited Vulnerabilities catalog. Federal agencies have strict deadlines to patch. [More info](https://www.cisa.gov/news-events/alerts/2026/04/28/cisa-adds-two-known-exploited-vulnerabilities-catalog)

- **Hugging Face AI Platform Flaw (CVE-2026-25874)**: A critical vulnerability could allow unauthorized access to private repositories or arbitrary code execution within the platform's infrastructure. [More info](https://thehackernews.com/2026/04/critical-cve-2026-25874-leaves-hugging.html)

- **Microsoft Entra ID Role Escalation**: An emergency patch addresses a logic error in Entra ID (formerly Azure AD) that allowed limited users to grant themselves administrative privileges. [More info](https://thehackernews.com/2026/04/microsoft-patches-entra-id-role-flaw.html)

## 🎯 Adversaries

- **BlueNoroff Hijacks Legitimate Entities**: The North Korean group is now using compromised infrastructure of previous victims to host malicious content, bypassing reputation filters to target financial institutions. [More info](https://www.darkreading.com/cyberattacks-data-breaches/bluenoroff-turns-victims-into-new-attack-lures)

- **LofyGang Targets Developers**: The Brazilian threat group has returned, spreading malicious NPM and GitHub packages that use advanced obfuscation to steal SSH keys and environment variables. [More info](https://thehackernews.com/2026/04/brazilian-lofygang-resurfaces-after.html)

- **Scattered Spider Member Charged**: US authorities filed charges against a key member of the group following an arrest in Finland. The group is known for aggressive SIM swapping and social engineering. [More info](https://www.bleepingcomputer.com/news/security/us-reportedly-charges-scattered-spider-hacker-arrested-in-finland/)

- **Ransomware "Turf War" (0APT vs. Krybit)**: Two rival groups are actively sabotaging each other's infrastructure and leaking data to damage reputations, creating a chaotic environment for victims. [More info](https://www.infosecurity-magazine.com/news/ransomware-turf-war-0apt-krybit/)

- **Firestarter Backdoor Targets Cisco**: A new Linux-based backdoor is specifically targeting Cisco Firepower devices to intercept network traffic and maintain long-term persistence. [More info](https://hackread.com/linux-firestarter-backdoor-cisco-firepower-devices/)

## 📈 Trends

- **Crypto Custody Concentration Risks**: Analysis shows $152.9 billion in digital assets is concentrated within top-tier custodians, creating a systemic "too big to fail" risk for the DeFi ecosystem. [More info](https://hackread.com/decoding-q1-2026s-152-9b-crypto-custody-concentration/)

- **OpSec Playbooks for EDR Evasion**: Investigations reveal threat actors are using detailed playbooks to mimic administrative behavior and use LoLBins to stay resident in memory and evade detection. [More info](https://www.bleepingcomputer.com/news/security/inside-an-opsec-playbook-how-threat-actors-evade-detection/)

- **Fake Captcha Mobile Scams**: Attackers are using fraudulent verification buttons to silently subscribe mobile users to expensive premium-rate SMS services via carrier billing systems. [More info](https://www.malwarebytes.com/blog/news/2026/04/fake-captcha-scam-turns-a-quick-click-into-a-costly-phone-bill)

- **Glassworm VS Code Extensions**: Malicious extensions in the VS Code Marketplace are mimicking popular tools to steal developer credentials and compromise supply chains. [More info](https://www.darkreading.com/application-security/fresh-glassworm-vs-code-extensions-supply-chain)

- **Robinhood Workflow Weaponized**: Attackers abused Robinhood's account creation process to send legitimate-looking phishing emails that successfully bypassed standard spam filters. [More info](https://www.bleepingcomputer.com/news/security/robinhood-account-creation-flaw-abused-to-send-phishing-emails/)

- **DHL Phishing Campaign**: A high-fidelity campaign mirrors the official DHL portal to harvest credentials through complex multi-stage attack chains and obfuscated JavaScript. [More info](https://hackread.com/dhl-phishing-scam-attack-chain-steal-passwords/)
