# Cybersecurity Newsfeed

## 📅 13/04/26

## 🛡️ Vulnerabilities

- **Adobe Acrobat and Reader Under Attack (CVE-2026-34621)**: Adobe released urgent updates for a critical use-after-free vulnerability being exploited in the wild. The flaw allows arbitrary code execution via crafted PDFs. Windows and macOS users are urged to patch immediately. [More info](https://securityaffairs.com/190697/security-adobe-fixes-actively-exploited-acrobat-reader-flaw-cve-2026-34621.html)

- **Critical Marimo Pre-Auth RCE (CVE-2026-39987)**: A critical vulnerability in the Marimo open-source Python notebook allows unauthenticated attackers to execute code via specially crafted websocket messages. Exploitation attempts have surged following a public PoC release. [More info](https://www.bleepingcomputer.com/news/security/critical-marimo-pre-auth-rce-flaw-now-under-active-exploitation/)

- **Iranian Threats to US ICS**: Nearly 4,000 industrial control systems in the U.S. are exposed to potential Iranian cyberattacks due to public internet reachability and outdated firmware. [More info](https://www.bleepingcomputer.com/news/security/nearly-4-000-us-industrial-devices-exposed-to-iranian-cyberattacks/)

## 🎯 Adversaries

- **"Hacker Claude" Breaches Mexican Government**: A threat actor allegedly exfiltrated 4.1 million sensitive records from a Mexican government portal using GPT-4-based scripts for reconnaissance and exploitation. [More info](https://hackread.com/hacker-claude-code-gpt-4-1-mexican-records/)

- **CPUID Supply Chain Attack Distributes STX RAT**: The infrastructure for CPU-Z and HWMonitor was compromised to distribute the STX Remote Access Trojan. Legitimate binaries were replaced with malware-laced versions to gain full system control. [More info](https://thehackernews.com/2026/04/cpuid-breach-distributes-stx-rat-via.html)

- **Glassworm Evolves with Zig-based Dropper**: The Glassworm threat actor is now using the Zig programming language to create low-detection droppers targeting developer tools and upstream repositories. [More info](https://securityaffairs.com/190638/malware/glassworm-evolves-with-zig-dropper-to-infect-multiple-developer-tools.html)

- **UAT-10362 Targets Taiwanese NGOs**: A new campaign is targeting non-governmental organizations in Taiwan with custom backdoors delivered via spear-phishing for geopolitical intelligence gathering. [More info](https://thehackernews.com/2026/04/uat-10362-targets-taiwanese-ngos-with.html)

- **Android Banking Trojan Hits Cambodia**: A sophisticated mobile threat is targeting Cambodian users, using accessibility services to bypass MFA and perform unauthorized fund transfers. [More info](https://hackread.com/android-banking-trojan-cambodia-scam-compounds/)

- **ClickFix Campaign Expands to macOS**: The "ClickFix" social engineering tactic now uses fake "missing font" errors to trick macOS users into executing malicious terminal commands that install infostealers. [More info](https://www.malwarebytes.com/blog/news/2026/04/clickfix-finds-new-way-to-infect-macs)

[More info](https://www.bleepingcomputer.com/news/google/google-rolls-out-gmail-end-to-end-encryption-on-mobile-devices/)
