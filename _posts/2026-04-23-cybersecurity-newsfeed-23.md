# Cybersecurity Newsfeed

## 📅 23/04/26

## 🛡️ Vulnerabilities

- **Critical ASP.NET Core RCE**: Microsoft has issued patches for a critical vulnerability in ASP.NET Core that could allow for unauthenticated remote code execution. The flaw stems from improper handling of specific request headers. [More info](https://thehackernews.com/2026/04/microsoft-patches-critical-aspnet-core.html)

- **Oracle April 2026 CPU**: Oracle’s April 2026 Critical Patch Update provides 365 new security patches. It addresses numerous critical vulnerabilities, including several with CVSS scores of 9.8 or higher in Database and WebLogic Server. [More info](https://blog.qualys.com/vulnerabilities-threat-research/2026/04/22/oracle-critical-patch-update-april-2026-security-update-review)

- **CISA Adds New Flaw to KEV**: CISA has added a new vulnerability to its Known Exploited Vulnerabilities (KEV) Catalog, affecting a widely deployed enterprise software component. Federal agencies are mandated to patch by a strict deadline. [More info](https://www.cisa.gov/news-events/alerts/2026/04/22/cisa-adds-one-known-exploited-vulnerability-catalog)

- **BridgeBreak ICS Vulnerabilities**: Critical flaws in serial-to-Ethernet converters from Lantronix and Silex Technology allow unauthenticated administrative access, potentially leading to full device compromise. [More info](https://securityaffairs.com/191114/hacking/critical-bridgebreak-flaws-impact-lantronix-and-silex-technology-converters.html)

- **iOS Notification Privacy Bug**: Apple released updates to fix an iOS bug that caused deleted notification data to persist in the system database, posing a risk for data leakage. [More info](https://www.bleepingcomputer.com/news/security/apple-fixes-ios-bug-that-retained-deleted-notification-data/)

## 🎯 Adversaries

- **"Gentlemen" Ransomware on the Rise**: A professionalized RaaS group using Rust-based encryptors is targeting mid-market enterprises in Europe and North America with double-extortion tactics. [More info](https://www.darkreading.com/threat-intelligence/gentlemen-rapidly-rise-ransomware)

- **Mustang Panda Targets Asia**: The threat actor is targeting government entities in India and South Korea using a new modular backdoor dubbed "LotusLite" delivered via spear-phishing. [More info](https://hackread.com/mustang-panda-india-s-korea-lotuslite-backdoor/)

- **Harvester Deploys Gogra Linux Backdoor**: This malware uses the Microsoft Graph API for C2 communications, allowing malicious traffic to blend in with legitimate cloud service activity. [More info](https://thehackernews.com/2026/04/harvester-deploys-linux-gogra-backdoor.html) | [Technical details](https://www.bleepingcomputer.com/news/security/new-gogra-malware-for-linux-uses-microsoft-graph-api-for-comms/)

- **Kyber Gang Adopts Post-Quantum Crypto**: The Kyber ransomware group is integrating PQC into its Windows encryptors to future-proof stolen data against quantum decryption. [More info](https://www.bleepingcomputer.com/news/security/kyber-ransomware-gang-toys-with-post-quantum-encryption-on-windows/)

- **Lotus Wiper Targets Venezuela**: A destructive malware campaign is overwriting MBRs and wiping files in Venezuelan organizations, prioritizing disruption over financial gain. [More info](https://thehackernews.com/2026/04/lotus-wiper-malware-targets-venezuelan)

- **Mirai Hits EOL D-Link Routers**: A new botnet campaign is exploiting RCE flaws in end-of-life D-Link hardware that no longer receives security patches. [More info](https://www.bleepingcomputer.com/news/security/new-mirai-campaign-exploits-rce-flaw-in-eol-d-link-routers/)

- **Worm-like npm Supply Chain Attack**: A sophisticated attack on the npm ecosystem uses self-spreading packages to steal authentication tokens from developers. [More info](https://www.bleepingcomputer.com/news/security/new-npm-supply-chain-attack-self-spreads-to-steal-auth-tokens/)

- **Iran 313 DDoS Hits Bluesky**: The social media platform sustained disruptions following a DDoS attack by a hacktivist group citing moderation grievances. [More info](https://hackread.com/bluesky-online-ddos-attack-iran-313-team/)
