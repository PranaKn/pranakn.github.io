# Cybersecurity Newsfeed

## 📅 17/04/26

## 🛡️ Vulnerabilities

- **CISA Adds Critical Flaw to KEV Catalog**: CISA has added a high-impact vulnerability to its Known Exploited Vulnerabilities catalog, mandating federal agencies to remediate the flaw. This move highlights active exploitation and the urgent need to patch to prevent unauthorized access. [More info](https://www.cisa.gov/news-events/alerts/2026/04/16/cisa-adds-one-known-exploited-vulnerability-catalog)

- **Microsoft Defender "RedSun" Zero-Day PoC**: A proof-of-concept exploit dubbed RedSun has been released, demonstrating a method to gain SYSTEM privileges on Windows. The flaw targets the antivirus engine's scanning process, allowing for arbitrary code execution. [More info](https://www.bleepingcomputer.com/news/microsoft/new-microsoft-defender-redsun-zero-day-poc-grants-system-privileges/)

- **Cisco Patches Four Critical Identity Vulnerabilities**: Cisco released fixes for severe flaws in its identity management solutions that could allow unauthenticated attackers to bypass authentication or gain administrative control by exploiting improper security token validation. [More info](https://thehackernews.com/2026/04/cisco-patches-four-critical-identity.html)

- **Fortinet FortiSandbox Command Injection**: Vulnerabilities CVE-2026-39813 and CVE-2026-39808 allow for command injection and unauthorized code execution. Given the role of sandboxes in malware analysis, these flaws pose a significant risk to secure environments. [More info](https://www.helpnetsecurity.com/2026/04/16/fortinet-fortisandbox-vulnerabilities-cve-2026-39813-cve-2026-39808/)

- **Cisco Webex Critical Session Flaw**: A vulnerability in Webex session token handling may allow unauthorized users to join private meetings. While some backend fixes are live, Cisco warns that certain configurations require manual administrator action. [More info](https://www.bleepingcomputer.com/news/security/cisco-says-critical-webex-services-flaw-requires-customer-action/)

- **Tails OS Local File Exposure**: A discovery in the Tails operating system reveals a flaw that could expose saved files to unauthorized access under specific conditions, undermining the isolation protocols intended for high-risk users. [More info](https://www.helpnetsecurity.com/2026/04/16/tails-vulnerability-expose-saved-files/)

## 🎯 Adversaries

- **North Korean "ClickFix" Targets macOS**: A social engineering campaign is targeting Mac users with fake browser "fix" scripts. Once executed, the scripts deploy malware to harvest credentials and system info, bypassing traditional macOS security perceptions. [More info](https://www.darkreading.com/application-security/north-korea-clickfix-target-macos-users-data)

- **Harmless Global "AV Killer" Campaign**: This campaign uses adware as a front to deploy a potent tool designed to disable security software. Once defenses are neutralized, attackers drop secondary payloads like ransomware or spyware. [More info](https://www.darkreading.com/cyberattacks-data-breaches/harmless-global-adware-av-killer)

- **PowMix Botnet Hits Infrastructure**: The newly identified PowMix botnet uses obfuscated PowerShell and mixed-language payloads to evade EDR systems, focusing on credential theft and persistent backdoor establishment. [More info](https://thehackernews.com/2026/04/newly-discovered-powmix-botnet-hits/)

- **Fake Slack Installers Deploy Hidden VNC**: Malicious campaigns are distributing fake Slack installers that install a hidden Virtual Network Computing (VNC) server, allowing attackers to remotely control systems via a stealthy desktop session. [More info](https://www.malwarebytes.com/blog/threat-intel/2026/04/a-fake-slack-download-is-giving-attackers-a-hidden-desktop-on-your-machine)

- **UAC-0247 Targets Ukrainian Healthcare**: Threat actor UAC-0247 is targeting medical clinics with phishing lures and malware designed for long-term espionage and sensitive data exfiltration in the healthcare sector. [More info](https://thehackernews.com/2026/04/uac-0247-targets-ukrainian-clinics-and/)

- **ZionSiphon Sabotages Water Systems**: A new malware strain, ZionSiphon, targets industrial control systems (PLCs) to manipulate water treatment parameters, posing a direct physical threat to public utility safety. [More info](https://www.bleepingcomputer.com/news/security/zionsiphon-malware-designed-to-sabotage-water-treatment-systems/)

## 📈 Trends

- **Google Gemini AI vs. Malicious Ads**: Google is expanding its use of Gemini AI to scan and block deceptive advertisements, leveraging large language models to identify malvertising patterns that traditional filters miss. [More info](https://www.bleepingcomputer.com/news/google/google-expands-gemini-ai-use-to-fight-malicious-ads-on-its-platform/)

- **ATHR Vishing Platform Automates Fraud**: The emergence of the ATHR platform allows attackers to use AI-driven voice agents to scale social engineering, using realistic interactive prompts to trick victims into revealing MFA codes. [More info](https://www.bleepingcomputer.com/news/security/new-athr-vishing-platform-uses-ai-voice-agents-for-automated-attacks/)

- **AI Coding Assistants Vulnerable to Prompt Injection**: Claude Code, Gemini CLI, and GitHub Copilot have been found susceptible to prompt injection via code comments in public repositories, potentially leading to data exfiltration or environment compromise. [More info](https://www.securityweek.com/claude-code-gemini-cli-github-copilot-agents-vulnerable-to-prompt-injection-via-comments/)
